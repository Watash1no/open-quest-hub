# Security Policy

## Supported Versions

We only provide security updates for the latest stable release.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Open Quest Hub, please do NOT create a public issue. Instead, report it privately through one of the following methods:

- **GitHub Private Vulnerability Reporting**: Use the "Security" tab on our GitHub repository to report a vulnerability.
- **Direct Contact**: If the above method is unavailable, please contact the maintainer directly through their GitHub profile.

Please include the following information in your report:
- A description of the vulnerability
- Steps to reproduce (proof of concept)
- Potential impact

We will acknowledge your report within 48 hours and provide a timeline for resolution.
